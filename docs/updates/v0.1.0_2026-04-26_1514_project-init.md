# v0.1.0 2026-04-26 15:14 初始化串口助手项目

本次更新创建 MY Port Assistant 的第一版基础项目，完成 C# WPF 串口助手的工程骨架、基础功能和构建验证。

## 修改目标

- 建立可上传 GitHub 的项目目录结构。
- 使用 C#、.NET 8、WPF 实现 Windows 串口助手第一版。
- 将 .NET SDK 和 NuGet 缓存优先放在 E 盘。
- 完成基础串口收发能力。

## 新增内容

- 新增 `README.md`、`LICENSE`、`.gitignore`、`NuGet.Config`、`global.json`。
- 新增解决方案 `MY_port.sln`。
- 新增 WPF 项目 `src/MYPortAssistant`。
- 新增构建脚本 `scripts/build.ps1`。
- 新增更新记录目录 `docs/updates`。
- 新增 `System.IO.Ports` 串口依赖。

## 已实现功能

- 串口列表扫描与刷新。
- 波特率、数据位、停止位、校验位配置。
- 打开串口和关闭串口。
- 接收区显示。
- ASCII 显示和 HEX 显示切换。
- 文本发送和 HEX 发送。
- 清空接收区。
- 接收字节统计。
- 发送字节统计。
- 状态栏显示当前操作和连接状态。

## 环境处理

- .NET SDK 8.0.420 已安装到 `E:\DevTools\dotnet`。
- NuGet 包缓存配置为 `E:\NuGetPackages`。
- 项目构建使用 `E:\DevTools\dotnet\dotnet.exe`。
