# v0.1.2 2026-04-26 21:23 发布自带运行环境版本并增加图标

本次更新解决开发版 exe 在未安装 .NET Desktop Runtime 的电脑上无法直接运行的问题，并为程序增加基础软件图标。

## 修改目标

- 发布自带 .NET 运行环境的 Windows x64 版本。
- 避免用户双击 exe 时提示安装 .NET Desktop Runtime。
- 为程序配置基础应用图标。
- 更新桌面快捷方式指向正式发布版 exe。

## 新增内容

- 新增图标文件：`src/MYPortAssistant/Assets/AppIcon.ico`。
- 新增正式发布目录：`publish/MYPortAssistant-v0.1.2/`。
- 新增本次更新记录文件。

## 修改内容

- 项目版本号从 `0.1.0` 更新为 `0.1.2`。
- 程序文件版本更新为 `0.1.2.0`。
- 在项目文件中配置 `ApplicationIcon`。
- 使用自包含发布方式生成正式版 exe。

## 发布命令

```text
E:\DevTools\dotnet\dotnet.exe publish .\src\MYPortAssistant\MYPortAssistant.csproj --configuration Release --runtime win-x64 --self-contained true -p:PublishSingleFile=true -p:EnableCompressionInSingleFile=true -p:IncludeNativeLibrariesForSelfExtract=true --output E:\MY_lunwen\MY_port\publish\MYPortAssistant-v0.1.2
```

## 验证结果

- 已生成正式版程序：`publish/MYPortAssistant-v0.1.2/MYPortAssistant.exe`。
- 程序大小约 71 MB。
- 已将桌面快捷方式 `MY Port Assistant` 和 `串口助手` 指向正式版 exe。
- 已启动正式版程序进行验证。
